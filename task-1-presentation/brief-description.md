# Task 1 – Digital Literacy Awareness Infographic

This task involves creating a one-page infographic to explain the concept of digital literacy and its importance for students.

The infographic covers four main areas: what digital literacy is, useful digital tools for students, safe internet practices and maintaining a professional online presence. It is designed to present information in a simple and clear visual format so that it is easy to understand.

The infographic was created using Canva and focuses on spreading awareness about using technology responsibly and effectively in daily life.
