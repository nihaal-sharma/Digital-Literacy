# Task 2 – Reflection Notes

For this task, I created and updated my profiles on platforms like GitHub, LinkedIn and Kaggle to build my digital portfolio. Each of these platforms serves a different purpose and is useful for students.

GitHub is mainly used to store and showcase coding projects. It helps in building a strong technical profile. LinkedIn is a professional networking platform where I can connect with other students, professionals and recruiters. Kaggle is useful for practicing data science and machine learning skills through datasets and competitions.

Setting up these profiles helped me understand the importance of having a professional online presence. I learned how to present my basic information in a clear and professional way.

Over the next four years, I plan to regularly update my GitHub with projects, improve my LinkedIn profile by adding skills and achievements and use Kaggle to practice and improve my knowledge in AI and Machine Learning. These platforms will help me build my skills and prepare for future internship and job opportunities.
